About the project

A lightweight recommendation prototype for cultural content in a target language.

Goal & MVP

Goal

Suggest music, films/series, and videos aligned with the learner's preferences in their target language.

MVP scope

Start with one content type (e.g., films) and one target language. Use genre similarity + popularity to retrieve a Top-5.

Two input modes

  • Filters quick, declarative preferences.
  • History optional upload for finer personalisation.
